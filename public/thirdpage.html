Tools

Express

Express er et web framework til Nodes.js.

Express et library af metoder der gør det nemt at udvikle Node baserede web applikationer.


Nodemon

Nodemon er et udviklingsværktøj som automatisk genstarter programmet hver gang der sker en ændring i koden.

Nodemon er smart fordi man ikke behøver at genstarte koden manuelt efter hver ændring.

npm nodemon app.js - kører programmet i Nodemon tilstand


Cross-env

Cross-env bruges til at sætte miljø variabler og parse dem til Node.js.

Normalt gøres det på forskellige måder på tværs af forskellige OS. Hvilket gør det problematisk hvis udviklingsholdet ikke arbejder i samme OS.


"scripts": { "start-dev": "cross-env NODE_ENV=DEV PORT=8080 nodemon app.js" },

Dette script er angivet i package.json og indeholder in cross-env command line som sætter miljø variablen til DEV(development), porten til 8080 og starter programmet i Nodemon
